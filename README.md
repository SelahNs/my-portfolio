# Personal Portfolio Website 💻🌐

This repository contains the source code for my personal portfolio website. It serves as a central hub showcasing my engineering background, featured software architectures, hardware designs, and full-stack technical capabilities.

### 🔗 [Live Portfolio Website](https://selehadin.netlify.app)

---

## 📸 Portfolio Features

This portfolio was designed from scratch to be exceptionally lightweight, performant, and modern, using zero framework dependencies.

*   **Interactive Terminal Emulator**: A custom vanilla JavaScript typewriter script that simulates a Zsh shell executing commands (`whoami`, `cat profile.json`, `ls ./projects`) to present my background in an engaging, interactive format.
*   **Tactile Grain Overlay**: Uses a custom, low-opacity SVG noise filter injected into the CSS `body::before` pseudo-element to create a high-end, subtle tactile texture across the dark-theme UI.
*   **Dynamic Scroll Reveal & Skill Bars**: Utilizes the modern HTML5 `IntersectionObserver` API to detect scroll intersections, smoothly animating visual sections into view and dynamically filling the tech stack skill bars.
*   **Semantic, Responsive Layout**: Built strictly with semantic HTML5 and modern CSS3 (Flexbox/Grid), maintaining complete responsiveness from mobile viewports up to widescreen displays.
*   **Integrated Contact Dispatcher**: Features a contact form integrated with Formspree's API to securely dispatch messages directly to my email inbox without leaving the page.

---

## 🛠️ Tech Stack

- **Markup & Styling**: HTML5, Custom CSS3 Variables, Google Fonts (Syne, DM Sans, DM Mono)
- **Scripting & DOM**: Vanilla JavaScript (ES6)
- **Integrations**: Formspree Mail API
- **Hosting & Deployment**: Netlify

---

## 📐 Design Philosophy

*   **Zero Framework Bloat**: Built purely with vanilla technologies to ensure near-instant load times, minimal bundle sizes, and optimal Google Lighthouse performance.
*   **Edge of Hardware & Code**: The visual styling reflects my hybrid background as an Electrical and Computer Engineering (ECE) student, bridging low-level hardware terminal designs with modern, flat-vector web interfaces.
