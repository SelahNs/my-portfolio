I'm Selehadin Nasir, an Electrical & Computer Engineering student at Addis Ababa University. Studying ECE has given me a grounded understanding of how computers work — from boolean logic and digital circuits up through memory architecture and operating systems.

CodeDash

A real-time developer analytics platform. A custom VS Code extension silently captures coding sessions — active language, file activity, session time — and syncs them over WebSockets to a Node.js/Express/MongoDB backend. Integrating with historical GitHub API data, a React dashboard renders live activity graphs and session history with Recharts.

Smart Logic Simulator

A native digital logic circuit simulator built in JavaFX. Features real-time gate evaluation, custom vector rendering via SVGPath, and custom binary serialization (*.ckt) to save and reload circuit files locally without a third-party simulation engine.
